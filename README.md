# Moodly Backend

Backend API for the Moodly mood tracking app.

## Technologies

- TypeScript
- Vercel (serverless)
- Supabase (database)

## Setup

1. `npm install`
2. Set environment variables: SUPABASE_URL, SUPABASE_ANON_KEY, MASTER_PASSWORD
3. Deploy to Vercel

## API Endpoints

- GET /api/get-entries
- POST /api/save-entry
- DELETE /api/delete-entry
- GET /api/get-analytics
- POST /api/verify-password
